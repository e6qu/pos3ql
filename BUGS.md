# Known bugs

Bugs are tracked here, not in code comments. Each entry gets a stable ID
(`B-NNN`), a repro (for simulator-found bugs: the seed and the exact command),
and the fixing commit once resolved. Update this file in the same commit
series as the phase that finds or fixes the bug.

See [docs/terminology.md](docs/terminology.md) for the glossary and naming
rules; architecture is in [README.md](README.md), the roadmap in
[PLAN.md](PLAN.md), and standing directives in [AGENTS.md](AGENTS.md).
